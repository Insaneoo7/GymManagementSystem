# Gym Management System

A Gym Management System built using .NET Clean Architecture with Entity Framework Core.

## Features
- Member Management
- Trainer Management
- Attendance Tracking
- Payment Management
- Diet Plan Management
- Workout Plan Management

## Architecture
This project follows Clean Architecture with separate layers:

- API
- Application
- Domain
- Infrastructure

## Technologies Used
- .NET 8
- Entity Framework Core
- SQL Server
- Clean Architecture
